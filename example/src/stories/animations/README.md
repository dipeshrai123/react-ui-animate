# Animation Examples

This directory contains comprehensive examples for all animation APIs in react-ui-animate.

## Structure

### Components

- **animate** - Basic animated components
- **makeAnimated** - Creating custom animated components

### Descriptors

- **withSpring** - Spring-based animations
- **withTiming** - Timing-based animations
- **withDecay** - Decay animations
- **withDelay** - Delayed animations
- **withSequence** - Sequential animations
- **withLoop** - Looping animations

### Drivers (Low-level)

- **timing** - Timing driver
- **spring** - Spring driver
- **decay** - Decay driver
- **parallel** - Parallel animations
- **sequence** - Sequential animations
- **loop** - Loop animations
- **delay** - Delay utility

### Utilities

- **Easing** - Easing functions
- **combine** - Combining AnimateValues
- **to** - Interpolation utility
- **clamp** - Clamping utilities

### Hooks

- **useValue** - Value hook examples

### Modules

- **Presence** - Enter/exit animations

### Recipes

- **All Recipes** - Pre-built animations

### State Animations

- **hover** - Hover state animations
- **press** - Press state animations
- **focus** - Focus state animations

### Advanced

- Complex examples combining multiple features
